# jiaotong

#### 名称
交通大学列车检修系统

#### 简介
这是一个用来检修列车部位时，根据姿态、手臂、手腕三个种类的各个动作组合出一套检修列车的组合动作，并以此动态来进行评分，并给出建议与存在的问题。此项目使用vue构建
